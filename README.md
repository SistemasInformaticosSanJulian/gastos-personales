## Proyecto Control de gastos personales
### Descripcion
Se desea crear una aplicacion web para el control de gastos personales

### Funcionalidades
#### Gestion de Perfil
- Atributos (nombres, apellidos, celular, correo)

#### Gestion de Categorias
- Atributos (nombre, descripcion)

Ejemplo: Comida, Transporte, Hogar, Ocio, Salud

#### Gestion de Gastos 
- Atributos (categoria, monto, fecha)
